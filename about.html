このサイトについて

サイト作成の目的

このサイトは、日本工学院専門学校Webプログラミング課題として作成した自己紹介サイトです。HTML5、CSS3を使用して、私自身について多角的に紹介することを目的としています。

また、学校で学んだWeb技術の知識を実践的に活用し、見やすく機能的なWebサイトを制作することで、今後のWeb開発スキル向上の基礎とすることも目指しています。

サイト構成について

このサイトは以下の5つのページで構成されています：

🏠 HOME（トップページ）

基本的な自己紹介と基本情報を掲載。サイト全体の入り口となるページです。

🎯 趣味ページ

私の趣味である音楽鑑賞、映画鑑賞、読書について詳しく紹介。時間配分や将来やってみたい趣味も掲載しています。

🎓 学校生活ページ

現在通っている日本工学院専門学校での学習内容、好きな授業、一日の流れなどを紹介しています。

✨ 将来の夢ページ

AIエンジニアになるという目標と、そのための具体的な学習計画、取得予定資格などを詳述しています。

ℹ️ このページ（ABOUT）

サイトの説明、使用技術、お問い合わせフォームを設置しているページです。

使用技術とデザイン

技術要素 使用箇所 説明
HTML5 Semantic Tags 全ページ header, main, article, aside, section, footer等を使用
CSS Grid Layout レイアウト全体 メインコンテンツとサイドバーの配置
Responsive Design 全ページ スマートフォンでも見やすいデザイン
Hover Effects ナビゲーション ユーザビリティ向上のためのインタラクション
Table Elements 各ページ データを整理して表示するために活用

特に工夫した点

  • ユーザビリティ - 直感的に操作できるナビゲーション設計
  • 情報の整理 - 表や段落を使って読みやすく構成
  • 視覚的な魅力 - 色彩バランスと余白を意識したデザイン
  • 一貫性 - 全ページで統一されたレイアウトとスタイル
  • アクセシビリティ - セマンティックHTMLとalt属性の適切な使用
